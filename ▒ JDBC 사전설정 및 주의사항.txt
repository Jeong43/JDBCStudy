■■■ JDBC 사전설정 및 주의사항 ■■■

○ 해당 워크스테이션에 오라클이 설치된 경우 (← 현재 학습 환경)

   - 시스템의 classPath 를 다음과 같이 생성 및 변경

     (11g Express Edition 설치 경로가 c:\oralcexe 인 경우...)
     .;C:\oraclexe\app\oracle\product\11.2.0\server\jdbc\lib\ojdbc6.jar

     (10g Enterprise/Standard Edition 설치 경로가 c:\oralcexe 인 경우...)
     .;C:\oracle\product\10.x.x\db_x\jdbc\lib\ojdbc14.jar

  -> 탐색기 - 내PC - 속성 - 고급시스템설정 - 환경변수 - 하단의 '시스템 변수'에서 CLASSPATH 가 없을 경우 -> 새로 만들기 클릭하여 -> 변수이름: 'CLASSPATH', 변수값: '.;C:\oraclexe\app\oracle\product\11.2.0\server\jdbc\lib\ojdbc6.jar' 입력 (가장 위에 있음...)


○ 해당 워크스테이션에 오라클이 설치되어 있지 않은 경우 (← 실무 환경)

   - Oracle 용 JDBC 드라이버를 다운로드 해야 한다.

     가. 다운로드 경로
	 ① https://www.oracle.com/database/technologies/appdev/jdbc.html
	    -------- → 보안 프로토콜 (http:// → 일반 프로토콜)
	 ② 페이지 중간의 'Get Started' 항목에서 'JDBC Download' 버튼을 클릭
	 ③ 현재(2021년 9월 27일 기준) 다운로드 제공하지 않음.

     나. 파일명
	 『ojdbc6.jar』 또는 『ojdbc14.jar』 파일

     다. 시스템의 classPath 에 변수값으로 ojdbc6.jar(ojdbc14.jar) 파일 추가(경로 포함)


※ 공통
   - JDBC 드라이버 설치를 위한 ojdbc6.jar(ojdbc14.jar) 파일을
     다음의 경로에 복사&붙여넣기

     가. C:\Program Files\Java\jdk1.8.0_281\jre\lib\ext 
	 (나는 D:\Program Files\Java\jdk1.8.0_281\jre\lib\ext)
     나. C:\Program Files\Java\jre1.8.0_281\lib\ext
	 (나는 D:\Program Files\Java\jre1.8.0_281\lib\ext)


※ JSP 실습을 위한 추가 설정
   - 웹 서버에서 실질적으로 Oracle 용 JDBC 드라이버를 찾는 경로는
     『아파치톰캣루트\lib』이기 때문에 『ojdbc6.jar』 파일을
     이 경로에 복사&붙여넣기 한다.

   - 아파치 톰캣 다운로드 경로 → http://tomcat.apache.org/
     : 홈페이지 좌측의 Download에서, Tomcat 버전을 선택 → Core 의 zip 파일을 다운

   	(우리는 '톰캣'이라는 웹 컨테이너를 사용하기 위해 설치함!)

   → apache-tomcat-8.5.54 를 압축 풀어서 C드라이브에 붙여넣기 
   → 『아파치톰캣루트\lib(C:\apache-tomcat-8.5.54\lib)』에 『ojdbc6.jar』 파일을 붙여넣기
